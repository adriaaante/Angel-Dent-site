> Креативы для Яндекс.Директа (РСЯ): фото-визуалы + видео + рекламные тексты.
> Картинки и видео названы по теме оффера и спарены по номеру (`NN-тема`).
> Стиль и правила для новых объявлений — в [`STYLE.md`](./STYLE.md).

## Важно про формат
Скачанные PNG — это **только фото-визуал (портрет, 1024×1024)**, без текста.
Текст оффера («Пенсионерам — скидка 10%…») в Директе задаётся отдельно
(заголовок + текст объявления), он **не «вшит» в картинку**. Поэтому ниже для
каждого визуала указан текст, с которым он шёл.
